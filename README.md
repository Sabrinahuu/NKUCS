# NKUCS 2025FALL Exam Paper


## **数据可视化**

个人还是非常推荐这门课的。

给分是：10分雨课堂选择题+40分4次小的实验+50分大作业（小组作业 3人）

这个icu上有较为详细的说明。

雨课堂习题可以参考上一届的题目，很多类似或者完全一样，或者用GPT搜。这部分得到了10/10。

第一次实验需要在实验课时间内写完老师给的五个可视化图表关于信达雅的评价，认真写的话还是有点赶的。另外讲有关信达雅的内容的课程可以认真听一听，和这个作业相关性还是比较大的。

其他实验在大模型的辅助下完成并不困难，建议是完成之后都给老师发飞书问问她的意见，然后有针对性的进行修改。老师一般回复也非常及时，人特别好！

大作业的话是小组作业，所以选择队员以及主题比较重要。还是做完了可以问问老师意见，包括前期的设想也可以问问意见。这部分可以参考一下上一届的优秀大作业。个人感觉我们这一届都做得蛮好的，所以记得卷一卷hh。
